## 变量初始化和赋值

### 初始化

C++中有6种变量初始化方法

```cpp
int a;         // 默认初始化
int b = 5;     // 拷贝初始化 将右值拷贝到左侧变量中
int c( 6 );    // 直接初始化 用于更有效的初始化复杂对象

// 列表初始化 (C++11引进) 
// 列表初始化的检查更严格，不允许缩小转换范围
int d { 7 };   // 直接列表初始化
int e = { 8 }; // 拷贝列表初始化
int f {};      // 值列表初始化

int width { 4.5 }; // error
int width = 4.5;   // 4
```

> 列表初始化又称为**统一初始化**
>
> 在引入列表初始化前，某些类型的初始化需要使用拷贝初始化，而其他类型的初始化则需要使用直接初始化。引入列表初始化是为了提供在大多数情况下都有效的更一致的初始化语法。
